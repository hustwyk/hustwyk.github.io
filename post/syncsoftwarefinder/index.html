Solved Problem: Files in my Onedrive/NutStore folder have no green tick check symbol

2020-01-16
macOS Extension
1171 words 3 mins read times read

Contents

    • 00 问题出现
    • 01 尝试各种重启重装
    • 02 上网搜，问客服啊
    • 03 总结

记一次解决macOS Catalina中Finder插件冲突的问题。

00 问题出现

之前在用Mojave的时候，给Finder装了几个插件：iCloud Drive[系统自带], OneDrive Finder Integration, Easy New File Extension, Dropbox Finder Extension, NutstoreFinderExtension.

当时他们都还相安无事。

之后系统升级到了Catalina，然后打开Finder之后，发现OneDrive、NutStore同步文件右下角的绿色小勾都不见了，我也不知道哪些文件是要永久留盘，哪些是被free up space的。。哎，这咋整呢。。

01 尝试各种重启重装

虽然这bug很奇怪，但是毕竟是Catalina，bug多一点也没啥，那就先老规矩重启重装大法。

  • 系统重启 => 无駄だ（mudada）
  • Finder重启 => 无駄だ
  • Onedrive重启 => 无駄だ
  • Nutstore重启 => 无駄だ
  • Onedrive重装（Appstore） => 无駄だ
  • Onedrive花样重装（官网安装包） => 无駄だ
  • Nutstore重装 => 无駄だ

02 上网搜，问客服啊

一筹莫展之际，无心再做别的事（其实是快过年心里有点躁动）

劳资遇到这种问题，别人说不定也遇到过，上网搜，结果就只有 Microsoft 官方的形式回答

Microsoft Community: No sync icons in Finder

这回答跟废话没啥差别。。。

那就去找客服吧。

然而巨硬的是填表单发送一个question，还得构思一篇英文短文，这思绪万千错乱的神经容不得这般折腾了，先把巨硬搁置。

再看坚果云的，在线客服直接聊问题。

图文交流，虽然一开始也让我重启之类的，但是一番沟通后

把问题锁定在了插件冲突上，此时我把 Easy New File Extension 关掉之后，坚果云文件夹下果然出现了熟悉的绿色小勾

但是我的 Onedrive 的文件还全部都坚持着原貌，然后我趁机问了一下客服有没有遇到过onedrive的问题

在她建议下我把插件们开开关关，把软件们开开关关，然后发现了重大突破：

  • 只打开 OneDrive Finder Integration 一个插件的时候，把onedrive重启可以看到onedrive文件夹下的绿色小勾。此时一定得把 Easy New File Extension 关掉。
  • 此时再打开 NutstoreFinderExtension 发现nutstore目录下的文件也都有了绿色小勾，而且onedrive目录下的小勾都还在。（窃喜
  • 之后把 iCloud Drive 和 Dropbox Finder Extension 都打开，也都没啥问题。
  • 最后打开了 Easy New File Extension 这个传说中引起冲突的开关，发现竟然也都没事，onedrive和nutstore的绿色小勾都还在。

03 总结

刚刚废话这么久其实可以总结成一点，就是 Easy New File Extension 就是冲突开关，这个插件不能早于其他插件打开，不然其他插件就会无法生效。

正确的做法就是：先开启别的插件（onedrive、nutstore），等它们都生效后再打开 EasyNewFile插件。
